# ✦ Infinity Saving - Save Your Dreams ✦
> **v2.13.4** | A modern, secure, and AI-assisted financial tracking web application.

Developed by **Zycresth** & **Fahmi-astronot**.

---

## 🚀 Overview
**Infinity Saving** is a lightweight, local-first web application designed to help users manage their savings goals with a touch of modern aesthetics and AI intelligence. This project focuses on **Privacy**, **User Experience**, and **Localization**.

## ✨ Key Features
* **🧠 Smart AI Assistant:** Integrated AI chat to provide financial insights and target predictions.
* **🌍 Multi-Language Support:** Full localization for **Indonesian**, **English**, and **Malay**.
* **🛡️ Security Focused:** Client-side hashing (Bcrypt), Anti-Brute Force, and Guest Mode features.
* **📊 Data Visualization:** Dynamic charts (Chart.js) to track progress visually.
* **🎨 Glassmorphism UI:** Neon-Red themed interface with a dedicated Navigation Rail for a sleek Linux-desktop feel.
* **💾 Data Sovereignty:** Full JSON Export/Import functionality—your data stays with you.

## 🛠️ Tech Stack
* **Core:** HTML5, CSS3 (Modern Flex/Grid), Vanilla JavaScript.
* **Libraries:** 
    * [Chart.js](https://www.chartjs.org/) (Data Visualization)
    * [Bcrypt.js](https://github.com/dcodeIO/bcrypt.js/) (Security/Hashing)
    * [Canvas-Confetti](https://github.com/catdad/canvas-confetti) (UX/Effects)
    * [jsPDF](https://github.com/parallax/jsPDF) & [html2canvas](https://html2canvas.hertzen.com/) (Reporting)
